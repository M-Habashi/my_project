To commit changes
- git add .
- git commit -m "commit letter"

To branch
- git checkout -b new_branch

To checkout to main
- git checkout main

To checkout to a commit
- git checkout commit_hash <- Never touch, just look

To merge a branch 
- git merge branch_name <- Make sure to be in main branch

To push changes
- git push

To pull changes
- git pull

To delete a branch
- git branch -d branch_name
